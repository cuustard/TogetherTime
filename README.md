# Together Time

A small React + Vite app for comparing two home time zones and finding the best time to call.

## Scripts

- `npm run dev` starts the local dev server.
- `npm run build` creates a production build.
- `npm run lint` checks the project with ESLint.
- `npm run preview` serves the production build locally.

## Notes

- Open the app with `?me=amy` to view it from Amy's perspective.
- Time blocks are defined in `src/App.jsx` and rendered directly in the main view.
